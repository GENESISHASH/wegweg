# wegweg

## getting started

using [npm](https://npmjs.org)

```
npm i wegweg --save
```

## example

``` javascript
// default init
var _ = require('wegweg')()

// with globals
var _ = require('wegweg')({globals:true})

// with globals and shelljs globals
var _ = require('wegweg')({globals:true,shelljs:true})
```
